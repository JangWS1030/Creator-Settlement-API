강사 선택

강사를 선택하면 아래 등록, 정산, 조회 기능이 모두 같은 강사 기준으로 맞춰집니다.

선택한 강사 강의 현황

판매 등록

현재 선택된 강사의 강의로 판매를 바로 등록합니다.

POST /api/sales 결과가 여기에 표시됩니다.

취소 등록

판매 조회 결과에서 선택한 판매 ID를 바로 취소 폼에 넣을 수 있습니다.

POST /api/sales/{saleId}/cancellations 결과가 여기에 표시됩니다.

월별 정산

연월만 선택하면 현재 선택된 강사의 월 정산을 조회합니다.

GET /api/settlements/monthly 결과가 여기에 표시됩니다.

판매 조회 / 운영자 집계

연월을 고르면 해당 월의 1일부터 말일까지 기준으로 판매 조회와 운영자 집계를 확인합니다.

조회 결과가 여기에 표시됩니다.

테스트 시나리오

버튼 한 번으로 시나리오 데이터를 생성 또는 재사용하고, 결과와 확인 SQL을 같이 봅니다.

시나리오 실행 결과와 SQL 확인 문구가 여기에 표시됩니다.
